# Jobs Data Repository

Stores job research data in CSV formats for freelance work, automations (n8n), lead generation, and agentic tasks.

## Structure

- `jobs/` - Job listings by category
- `sources/` - Source URLs and research notes

## Categories

- `freelance/` - Freelance job listings
- `automation/` - Automation opportunities (n8n, Zapier, etc.)
- `leadgen/` - Lead generation tasks
- `agentic/` - AI agent task opportunities
